# 🐳 Flask Docker Addition App

A simple web application built with **Flask, HTML, and CSS** that adds two numbers.  
The application is containerized using **Docker** for easy deployment.

---

## 📌 Features

- Simple UI for adding two numbers
- Flask backend
- Clean HTML & CSS design
- Docker container support
- Easy to run anywhere using Docker

---

## 🛠 Technologies Used

- Python
- Flask
- HTML
- CSS
- Docker

## 🚀 Run the Project Using Docker

### 1️⃣ Clone the repository

git clone https://github.com/YOUR_USERNAME/flask-docker-addition-app.git

### 2️⃣ Go to project folder

cd flask-docker-addition-app

### 3️⃣ Build Docker image

docker build -t flask-addition-app .

### 4️⃣ Run Docker container

docker run -p 5000:5000 flask-addition-app

### 5️⃣ Open in browser

http://localhost:5000
